# BETIVO

**BETIVO** is a lightweight, modular web app for social gaming and micro‑transactions. It combines a responsive UI, modular game engines, and a real‑time backend to deliver a fast, engaging experience for players and operators.

## Key features
- Modular game architecture (blackjack, roulette, plinko, precision)
- Real‑time leaderboard and player state via Firebase
- Audio and SFX manager with playlist support
- Shop, inventory, and cosmetic systems
- Responsive, accessible UI with reduced‑motion support
- Unit tests for core game logic and CI-ready tooling
